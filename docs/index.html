Purpose-built to bridge CrewAI's execution model with Chainlit's real-time chat UI — with a minimal, composable API.

🪜

Step rendering

Mirrors crew, task, agent, and tool events into Chainlit's native step tree. See exactly what your crew is doing, in real time.

💬

Human-in-the-loop

ChainlitAskUserTool lets any CrewAI agent pause execution and ask the user follow-up questions through chat.

Async kickoff

ChainlitCrewAdapter wraps crew.kickoff() with full async support — no blocking, no workarounds needed.

Trace control

Toggle show_agent_steps=False to suppress agent-level noise and keep traces clean for production use.

Quick start

Up and running
in minutes.

Two core patterns — step rendering for visibility, ask-user for interactive workflows.

Step rendering
Ask-user flow
your_app.py 
import chainlit as cl
from chainlit_crew_adapter import ChainlitCrewAdapter
from your_crew import build_my_crew

@cl.on_message
async def main(message: cl.Message) -> None:
    crew = build_my_crew()

    adapter = ChainlitCrewAdapter(
        cl=cl,
        crew=crew,
    )

    result = await adapter.kickoff(
        inputs={"user_request": message.content}
    )

    await cl.Message(content=str(result)).send()
your_app.py 
import chainlit as cl
from chainlit_crew_adapter import (
    ChainlitAskUserTool,
    ChainlitCrewAdapter,
)
from your_crew import build_ask_user_crew

@cl.on_message
async def main(message: cl.Message) -> None:
    ask_user_tool = ChainlitAskUserTool(
        timeout_seconds=180,
        author="Crew Assistant",
    )

    crew = build_ask_user_crew(ask_user_tool=ask_user_tool)

    adapter = ChainlitCrewAdapter(
        cl=cl,
        crew=crew,
        show_agent_steps=False,  # cleaner trace
    )

    result = await adapter.kickoff(
        inputs={"user_request": message.content}
    )

    await cl.Message(content=str(result)).send()

API reference

Two classes.
Clean interface.

ChainlitCrewAdapter

Runs your crew and mirrors events to Chainlit steps

cl
SupportsChainlit
The Chainlit module instance
crew
Crew
Your CrewAI crew instance
inputs
Mapping | None
Optional default inputs for kickoff
show_agent_steps
bool = True
Toggle agent-level step rendering
await kickoff()
async method
Runs crew and streams steps. Accepts optional inputs override.

ChainlitAskUserTool

Gives agents the ability to pause and ask the user questions

timeout_seconds
int = 120
How long to wait for a user response
author
str
Display name shown in the chat UI
fail_on_timeout
bool = False
Raise an error or gracefully continue on timeout
ask_user
callable | None
Override for custom ask-user implementation
chainlit_context
context | None
Pass explicit Chainlit context if needed
